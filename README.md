# NAU Student Platform

A modern web application for NAU students to share reviews, browse student-created guides, and access tools that make campus life easier.

> Note: This project is still in active development. Features and UI may change.

---

## Features

- 🧑‍🏫 Professor & course reviews  
- 📚 Student playbooks / guides  
- 💬 Discussions & Q&A between students  
- 🔐 Email + password authentication  
- 📱 Responsive, mobile-friendly layout  

---

## Tech Stack

- [React](https://react.dev/)  
- [TypeScript](https://www.typescriptlang.org/)  
- [Vite](https://vitejs.dev/)  
- [Tailwind CSS](https://tailwindcss.com/)  
- [shadcn/ui](https://ui.shadcn.com/)  
- [Supabase](https://supabase.com/) for auth & database  

---

## Getting Started

### Prerequisites

- Node.js (LTS recommended, e.g. 18+)
- npm or pnpm or yarn

### Installation

```bash
# 1. Clone the repository
git clone <YOUR_REPO_URL>

# 2. Go into the project folder
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
